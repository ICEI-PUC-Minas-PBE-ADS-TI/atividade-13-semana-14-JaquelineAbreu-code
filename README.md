### 🎬 Descrição do Projeto

Este projeto implementa um **Dashboard Dinâmico e Responsivo** para visualização e análise de dados de filmes. Através de uma interface moderna e intuitiva, o sistema apresenta informações catalogadas em três dimensões principais:

1. **Indicadores Principais** — Cards informativos que exibem:
   - Total de filmes cadastrados
   - Média de notas/avaliações
   - Filme mais recente na base de dados

2. **Visualizações Gráficas** — Utilizando Chart.js, são gerados:
   - **Gráfico de Pizza**: Distribuição de filmes por gênero
   - **Gráfico de Barras**: Distribuição das notas por faixa de classificação
   - **Gráfico de Linhas**: Lançamentos de filmes por ano

3. **Tecnologia Backend** — JSON Server oferece uma API REST fake para:
   - Servir dados persistidos em `db/db.json`
